Why Shortcut.css?

Everything you need to build modern web interfaces

🎯

Utility-First

Compose designs directly in your HTML with intuitive utility classes. No need to write custom CSS.

🪶

Lightweight

Only ~50KB unminified, ~15KB minified. Zero dependencies. Just pure CSS optimized for performance.

🚀

Modern

Built with modern CSS standards. Flexbox, Grid, custom properties, and more. No obsolete code.

💪

TypeScript Ready

Full TypeScript definitions for IntelliSense support. Get autocomplete for all utility classes.

🌙

Dark Mode

Built-in dark mode support with automatic system preference detection. Toggle with a single class.

🔧

Customizable

Easy theming with CSS custom properties. Override defaults without modifying the source code.

Quick Start

Get up and running in seconds

Install via NPM

npm install shortcut-css

Or use CDN

<link rel="stylesheet" 
      href="https://unpkg.com/shortcut-css@latest/dist/shortcut.min.css">

Start Building
